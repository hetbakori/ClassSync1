# ClassSync - Smart Campus Portal 🎓✨

ClassSync is a modern, unified ecosystem designed to give students real-time access to their academic lives. Built as a Progressive Web App (PWA), it allows students to install the portal directly to their mobile devices for a native app experience. 

The backend is entirely serverless, utilizing a live Google Sheets CSV to manage and push dynamic data to the student dashboard instantly.

## 🚀 Key Features

* **Live Timetable:** Automatically tracks the current time to display ongoing and upcoming lectures, including specific batch splits and room numbers.
* **Academic Insights:** Dynamic target-to-pass calculations showing exactly what scores are needed in upcoming mid-terms to clear subjects.
* **Real-Time Attendance:** A visual tracker for aggregate attendance that updates alongside the live database.
* **PWA Ready:** Fully installable on iOS and Android devices directly from the browser, functioning offline with service worker caching.
* **Premium UI/UX:** Features a dark ambient design, glassmorphism card layouts, magnetic buttons, and custom CSS snowfall animations.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Custom properties, animations, media queries)
* **Logic:** Vanilla JavaScript (ES6+)
* **Backend/Database:** Google Sheets (Published as CSV)
* **Libraries:** [PapaParse](https://www.papaparse.com/) for lightning-fast frontend CSV parsing
* **Icons:** [Phosphor Icons](https://phosphoricons.com/)

## 📱 How to Use

1. Navigate to the live GitHub Pages link.
2. If prompted by your mobile browser, select **"Add to Home Screen"** to install the app.
3. Open the portal and enter your **First Name**, **Last Name**, and **Roll Number**.
4. Press Enter or click "Enter Workspace" to authenticate against the live campus database.
5. Navigate via the bottom tabs to view your Today, Weekly, Results, and Attendance metrics.
